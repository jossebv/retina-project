◆ _timer_rx_setup()

void _timer_rx_setup ( )

Configure the timer tick. This timer sets the basis for tick counting for checking received NEC symbols.

This timer does not generate interruptions. It is like the Systick, but faster. The Systick has a period of 1 ms which is to large for the identification of NEC symbols. That is why we need another counter.

With this timer we are not interested very mucho on the value of the ARR register but on the PSC register. We will set up the ARR register to its maximum value to ensure that each increment of the prescaler corresponds to NEC_RX_TIMER_TICK_BASE_US. On the other hand, we set the PSC so that each increment of this register represents NEC_RX_TIMER_TICK_BASE_US.
